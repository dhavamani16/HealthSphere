

A full-stack medical/doctor management system built using React, Node.js, and MongoDB.

---

##  Features

- Doctor listing with sample data
- Local doctor seeding for testing
- Admin & client modules
- Clean UI using React + Vite

---

##  Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
